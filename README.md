## 介绍： 

#### 安卓查壳  支持：腾讯、梆梆、百度等16种壳

## 安装方法：

#### 使用Python3 , 执行 pip install -r requirements.txt   
#### 运行FlaskServer.py访问 http://127.0.0.1:80
#### 在windows10上测试通过。
